Ohjelmisto teknologioita

Tämä projekti sisältää hahmolistan, johon voi halutessaan lisätä uusia hahmoja.

Ohjeet kirjautumiseen:

Käyttäjätunnus: admin Salasana: admin (vain tällä tunnuksella on mahdollista poistaa hahmolistasta hahmoja)
Käyttäjätunnus: user Salasana: user (tällä tunnuksella on mahdollista katsella sekä lisätä hahmolistaan uusia hahmoja)
